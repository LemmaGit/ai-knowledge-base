# ai-knowledge-base
AI-powered knowledge base platform with article browsing, real-time search, clean documentation pages, and a modular React + Node.js architecture.

## Features
- Real-time article search
- Category-based browsing
- Clean, responsive UI
- REST API for articles and categories
- Markdown-friendly article content
- Organized full-stack structure (React + Node.js)

## Tech Stack
**Frontend:** React, React Router, TailwindCSS  
**Backend:** Node.js, Express
